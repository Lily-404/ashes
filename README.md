# 灰烬 (Ashes)

让你写下秘密，点击焚烧，将其化为灰烬，彻底消失。

"灰烬"是一个帮助人们释放心理负担的私密空间。在这里，你可以写下任何想要释放的秘密或想法，然后通过焚烧按钮将其化为灰烬，象征性地将这些负担从心中释放出去。

## 技术栈

- Next.js 15
- React 19
- TypeScript
- Tailwind CSS

## 本地开发

### 安装依赖

```bash
npm install
```

### 运行开发环境

```bash
npm run dev
```

### 构建生产版本

```bash
npm run build
```
